
# Yakuake sessions launcher

```bash
ln -s /path/to/cloned/repository/yakuake-sessions/main.py ~/.local/bin/yakuake-sessions
ln -s /path/to/cloned/repository/yakuake-sessions/config.json ~/.yakuake-sessions.json

# start one session
yakuake-sessions --run ncmpcpp
# or two sessions
yakuake-sessions --run ncmpcpp,htop
# or more!

```
